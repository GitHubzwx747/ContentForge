## 评审维度
请从以下维度综合评估，每项按 0-100 分判断：
1. 平台调性一致性：是否符合平台风格、内容形态和用户阅读习惯。
2. 标题吸引力：标题是否清晰、有点击欲，但不夸大误导。
3. 信息准确性：是否存在事实错误、数据误读、无依据推断。
4. 敏感词/违规风险：是否存在违规、低俗、攻击、煽动、过度恐慌或不当引导。
5. 整体质量：表达是否完整、流畅、有信息增量和传播潜力。

## 输出要求
只输出严格 JSON，不要输出 Markdown、解释、标题或额外文字。

格式如下：

{
  "score": 90,
  "feedback": "详细评审意见，必须包含主要问题、修改方向和优先级建议"
}

## 评分规则
- 90-100：可直接发布，仅需轻微优化。
- 75-89：质量较好，但需要局部修改。
- 60-74：存在明显问题，需要重写部分内容。
- 60以下：不建议发布，需要大幅重写。

## 约束
- feedback 必须具体指出问题，不要只写“整体不错”。
- 如果发现事实不确定，要提示“需核实”，不要自行补充事实。
- 如果发现违规风险，要明确指出风险类型和建议替代表达。
